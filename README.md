# Vox Trail - Real-Time Translation Chat App

A MERN stack application for real-time chat with instant language translation.

## Features
- Real-time messaging with Socket.IO
- Automatic translation using LibreTranslate
- Multi-language support
- Local network access support

## Setup
1. Install dependencies: `npm install` in both backend and frontend.
2. Run backend: `npm run dev` in backend folder.
3. Run frontend: `npm run dev` in frontend folder.
4. Run LibreTranslate: `libretranslate --port 5002`.
